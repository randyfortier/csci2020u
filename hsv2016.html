Round 1: Futuristic User Interfaces

  • Gesture-based input devices:
    • How does it work?
      • Images from 2+ cameras can be combined to get a rough 3D shape
      • These 3D shapes are turned into skeletal models
      • The change in 3D shape of these skeletal models over time is matched to certain patterns
        • e.g. How close to each other are the points in the skeleton in the viewed model to the reference model for a gesture?

Round 1: Futuristic User Interfaces

  • Holographic output devices:
    • How does it work?
      • Multiple sources of light are emitted
      • The light overlaps and interferes where we want a voxel (~pixel) to be
      • The actual spatial position of the object matches its apparent position
